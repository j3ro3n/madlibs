Want to set a timer?

This will prevent your rounds from lasting forever. If you set a timer, the round will automatically end when the timer runs out. Beware not to set this too low though! If you do not want to play with a timer, you may set this to 0 seconds.

Please be aware that only the first timer set for a session will be kept into account. All subsequent timers will be summarily ignored.
